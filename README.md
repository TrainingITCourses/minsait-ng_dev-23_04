## Installation and new generation

```bash
node -v
# Install Angular CLI
npm install -g @angular/cli
# Generate a new project
ng new your-name
```

## Use v12 clone for Node < 16

```bash
cd ng-v12
npm install
npm start
```

cd ng-v14
